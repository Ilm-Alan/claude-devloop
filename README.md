# Claude Devloop Marketplace

A Claude Code plugin marketplace containing iterative development tools.

## Installation

```bash
/plugin marketplace add Ilm-Alan/claude-devloop
/plugin install devloop@claude-devloop
```

## Plugins

### devloop

Run Claude in a while-true loop with the same prompt until task completion.

See [devloop/README.md](devloop/README.md) for full documentation.
